# 🌤 Weather App

A modern, responsive weather application built using **HTML, CSS, and Vanilla JavaScript**.  
The app fetches real-time weather data and a 5-day forecast using the **OpenWeather API**.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡 Current temperature, weather condition
- 💧 Humidity & 🌬 wind speed
- 📅 Current date display
- 📊 5-day daily forecast
- 🎨 Dynamic weather icons
- 🧊 Glassmorphism UI design
- 📱 Mobile-friendly layout

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **OpenWeather API**
